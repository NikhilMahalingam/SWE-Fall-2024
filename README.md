# SWE-Fall-2024
ECE 452 SWE Project

To build the client:
1. `cd client`
2. `npm install` to install all required modules.
3. `npm run build` to run the build.

To start the server:
1. `cd server`
2. `npm install` to install all required modules.
3. `npm start` to start the server. This should now be running on localhost:8000.
